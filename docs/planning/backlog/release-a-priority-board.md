# Release A Priority Board

## 우선순위 기준

- `P0`
  - Release A 성패를 좌우하는 필수 항목
- `P1`
  - 릴리즈 품질을 크게 높이는 핵심 항목
- `P2`
  - 있으면 좋지만 다음 릴리즈로 일부 이월 가능
