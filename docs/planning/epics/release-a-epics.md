# Release A Epics

## Epic A1. Codex Control Reliability

### 목적

Codex를 붙잡고, 다시 찾고, 안정적으로 입력하는 기반을 완성합니다.

### 주요 산출물

- 창 타겟 탐색기
- 타겟 락킹 정책
- 입력창 캘리브레이션 플로우
- 전송 전 확인용 프롬프트 프리뷰

### 포함 티켓

- `A-001`
- `A-002`

## Epic A2. Session & Prompt Operations

### 목적

프로젝트 정의와 단계 큐를 안정적으로 관리하고, 항상 “지금 무엇을 보내는지” 분명하게 만듭니다.

### 주요 산출물

- 세션 스키마
- 단계 포인터
- 프롬프트 조합기
- 저장/복원 흐름

### 포함 티켓

- `A-003`
- `A-004`

## Epic A3. Observation & Evidence

### 목적

Jarvis의 행동이 감(感)이 아니라 근거에 기반하도록 캡처, 메타데이터, 안정화 판단을 체계화합니다.

### 주요 산출물

- 캡처 파이프라인
- 증거 저장소
- 유사도/안정화 규칙
- false positive 감소 로직

### 포함 티켓

- `A-005`
- `A-006`

## Epic A4. Operator Trust & Safety

### 목적

자동화가 빨리 돌아가는 것보다 상단장님이 안심하고 맡길 수 있는 상태를 만듭니다.

### 주요 산출물

- 운영 대시보드
- 액션 타임라인
- 긴급 정지
- 재시도/보류/중단 가드레일
- 릴리즈 스모크 체크리스트

### 포함 티켓

- `A-007`
- `A-008`
- `A-009`
- `A-010`
