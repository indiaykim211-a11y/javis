# A-007: 운영 대시보드와 액션 타임라인

## 목적

상단장님이 로그를 깊게 읽지 않아도 현재 상태와 최근 행동을 한눈에 이해할 수 있게 만듭니다.

## 배경

자동화는 돌아가는데 상태가 안 보이면 불안합니다. javis는 “잘 보이는 운영석”이어야 합니다.

## 사용자 스토리

상단장님으로서, 지금 어떤 단계이며 방금 무슨 행동을 했고 다음에는 뭘 하려는지 바로 파악하고 싶습니다.

## 범위

- 현재 상태 카드
- 현재 단계 표시
- 최근 액션 타임라인
- 최근 캡처 목록
- 다음 액션 후보 표시
- 경고/주의 배지

## 제외

- 고급 차트
- 멀티 프로젝트 대시보드

## 구현 메모

- UI는 정보가 많아도 우선순위가 선명해야 함
- “지금 상태”, “최근 행동”, “다음 행동” 세 축을 항상 유지
- 타임라인은 사람이 개입한 액션을 구분 표시

## 완료 조건

- 현재 상태를 3초 안에 파악 가능
- 최근 행동 이력을 시간순으로 확인 가능
- 다음 액션 후보가 항상 보임

## 산출물

- 운영 콘솔 v1
- 액션 타임라인 컴포넌트
- 경고 배지 시스템
