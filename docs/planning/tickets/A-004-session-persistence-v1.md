# A-004: 세션 저장/복원 v1

## 목적

프로젝트 설정, 단계 상태, 캘리브레이션 값, 최근 타겟 정보를 끊김 없이 이어가게 만듭니다.

## 배경

Jarvis는 한 번 켜서 잠깐 쓰는 툴이 아니라, 프로젝트 맥락을 가지고 며칠씩 이어갈 수 있어야 합니다.

## 사용자 스토리

상단장님으로서, 앱을 껐다 켜도 지난 설정과 진행 상태가 그대로 복원되길 원합니다.

## 범위

- 세션 스키마 정리
- 저장 버전 필드 추가
- 최근 프로젝트 목록
- 마지막 타겟 정보 저장
- 단계 포인터 저장
- 캡처/로그 참조 저장

## 제외

- 장기 메모리 추천 시스템
- 원격 동기화

## 구현 메모

- v1은 파일 기반 저장으로 시작해도 되지만, 로그/증거 연결이 커질 경우 SQLite 전환을 열어둠
- 스키마 버전이 바뀌어도 마이그레이션 가능하게 설계

## 완료 조건

- 앱 재실행 후 세션이 정상 복원됨
- 이전 캘리브레이션과 단계 포인터를 재사용 가능
- 저장 구조가 이후 기능 확장에 무리가 없음

## 산출물

- 세션 스키마 v1
- 저장/복원 로직
- 최근 프로젝트 진입점
