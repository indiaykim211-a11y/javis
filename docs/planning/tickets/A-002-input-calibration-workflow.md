# A-002: 입력창 캘리브레이션 워크플로우

## 목적

Codex 입력창의 상대 좌표를 상단장님이 쉽게 맞추고 검증할 수 있게 만듭니다.

## 배경

Release A에서는 아직 시맨틱 입력창 인식보다 상대 좌표 의존도가 높습니다. 따라서 캘리브레이션 UX가 실전 사용성을 좌우합니다.

## 사용자 스토리

상단장님으로서, 코드를 건드리지 않고도 입력 위치를 맞추고 바로 테스트해 보고 싶습니다.

## 범위

- 캘리브레이션 모드 진입
- 클릭 위치 캡처
- 창 기준 상대 좌표 계산
- 좌표 미리보기
- 테스트 전송 버튼
- 저장/재적용

## 제외

- 입력창 자동 탐지
- OCR 기반 위치 추천

## 구현 메모

- “현재 마우스 위치 저장” 또는 오버레이 클릭 캡처 방식 검토
- 좌표는 창 크기와 함께 저장하여 재사용 시 보정 가능하게 설계
- 테스트 전송은 짧은 안전 문구 또는 submit off 모드로 지원

## 완료 조건

- 비개발자도 1분 내 입력창 위치를 설정 가능
- 테스트 전송으로 성공 여부를 확인 가능
- 세션 재실행 후에도 좌표가 유지됨

## 산출물

- 캘리브레이션 패널
- 좌표 저장 구조
- 안전 테스트 전송 플로우
