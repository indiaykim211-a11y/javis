# A-008: 긴급 정지와 안전 모드

## 목적

Jarvis가 잘못된 행동을 할 위험이 보이면 즉시 멈추고, 위험도가 높을 때는 자동으로 보수적인 모드로 내려오게 합니다.

## 배경

실전 운영에서 가장 중요한 신뢰 요소는 “언제든 끊을 수 있음”입니다. 긴급 정지는 핵심 신뢰 장치입니다.

## 사용자 스토리

상단장님으로서, 이상하다고 느끼는 순간 즉시 자동화를 멈추고 더 이상의 입력을 막고 싶습니다.

## 범위

- 글로벌 긴급 정지 버튼
- 단축키 기반 긴급 정지
- 안전 모드 상태 전환
- 위험 상태에서 자동 전송 차단
- 사용자 알림 문구

## 제외

- 원격 승인
- 음성 “멈춰” 인식

## 구현 메모

- 긴급 정지는 어떤 스레드 상태에서도 최대한 즉시 반영
- 안전 모드는 수동 승인 없이는 전송하지 않음
- 위험 사유를 코드와 문구 양쪽으로 남김

## 완료 조건

- 버튼 또는 단축키로 즉시 멈춤
- 안전 모드 진입 후 자동 전송이 차단됨
- 멈춘 이유가 UI와 로그에 명확히 보임

## 산출물

- 긴급 정지 플로우
- 안전 모드 상태 머신
- 위험 알림 UI
