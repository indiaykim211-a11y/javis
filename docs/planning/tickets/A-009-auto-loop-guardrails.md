# A-009: 자동 루프 가드레일

## 목적

javis가 자동 루프 중 잘못된 상태로 빠졌을 때 계속 밀어붙이지 않고, 적절히 멈추고 상단장님을 호출하도록 만듭니다.

## 배경

자동화는 성공할 때보다 실패할 때의 행동이 더 중요합니다. 실패 설계가 약하면 실제 사용이 어렵습니다.

## 사용자 스토리

상단장님으로서, javis가 이상 상태를 반복할 때 알아서 멈추고 보고해 주기를 원합니다.

## 범위

- 재시도 횟수 제한
- 타겟 상실 시 중단
- 일정 시간 이상 무변화 시 보류
- 연속 실패 시 안전 모드 전환
- 사용자 호출 메시지

## 제외

- 자동 보정 프롬프트 생성
- OCR 기반 실패 분류

## 구현 메모

- 실패 유형을 최소한 아래처럼 나눔
  - target_lost
  - send_failed
  - unstable_timeout
  - repeated_false_positive
- 실패 횟수와 최근 성공 여부를 함께 판단

## 완료 조건

- 무한 루프가 방지됨
- 반복 실패 시 자동으로 보수 모드로 전환됨
- 왜 멈췄는지 로그와 UI에서 이해 가능

## 산출물

- 자동 루프 가드레일 정책 v1
- 실패 사유 코드 체계
- 사용자 호출 메시지
