# Sprint 3: Polish and Release Readiness

## 스프린트 목표

Assistant Alpha를 “동작하는 프로토타입”이 아니라 “상단장님이 실제로 만져볼 수 있는 제품” 수준으로 마감합니다.

## 포함 티켓

- `R1-007` 컴팩트 비주얼/동작 다듬기
- `R1-008` Release 1 스모크 스위트

## 상세 목표

### 1) 컴팩트 제품감 확보

- 팝업 크기와 여백 조정
- 시각적 위계 정리
- 작은 해상도에서의 동작 점검

### 2) 제품 완성도 보정

- 팝업과 Control Center 전환 흐름
- 저장/미저장 상태 노출
- 어색한 개발자용 문구 제거

### 3) 릴리즈 체크리스트 확보

- 첫 실행
- 세션 복원
- 팝업 동작
- Control Center 진입
- 핵심 버튼 흐름

반복 검증 가능한 스모크 세트 마련

## 완료 조건

- 팝업이 작지만 답답하지 않음
- 설정창과 팝업 왕복이 자연스러움
- Release 1 체크리스트로 핵심 흐름을 빠르게 점검 가능

## 데모 시나리오

1. 첫 실행
2. 팝업 상태 확인
3. Control Center 진입
4. 최근 프로젝트 복원
5. 팝업 복귀
6. 스모크 시나리오 수행

## 주요 리스크

- 시각 다듬기가 끝없이 늘어날 수 있음
- 스모크 체크리스트가 실제 사용 흐름을 충분히 못 담을 수 있음
