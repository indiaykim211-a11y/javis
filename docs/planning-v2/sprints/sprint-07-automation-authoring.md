# Sprint 7: Automation Authoring

## 스프린트 목표

현재 프로젝트에 맞는 automation mode를 고르고, Codex 앱에 바로 옮길 수 있는 thread / project automation 초안을 만들게 합니다.

## 포함 티켓

- `R3-001` automation mode selector
- `R3-002` thread automation composer
- `R3-003` project automation composer

## 상세 목표

### 1) automation 필요 여부부터 판단

- no automation
- thread automation
- standalone / project automation

을 같은 화면에서 비교할 수 있어야 합니다.

### 2) Codex 앱 handoff 준비

- prompt
- cadence
- worktree
- safety note

를 한 번에 보여줄 수 있어야 합니다.

## 완료 조건

- 상단장님이 현재 프로젝트에서 automation이 필요한지 아닌지 바로 판단 가능
- thread / project automation 초안을 Codex 앱으로 쉽게 옮길 수 있음

## 스프린트 산출물

- Automation Mode Selector v1
- Thread Composer v1
- Project Composer v1
