# Sprint 16: Voice Foundation

## 목표

음성을 `입력 이벤트`로 받아서 `구조화 intent 후보`까지 만드는 기반을 올립니다.

## 포함 티켓

- R6-001 push-to-talk capture shell
- R6-002 transcription and command normalizer
- R6-003 voice intent router

## 완료 조건

- push-to-talk 시작 / 종료 상태가 제품 표면에 보임
- transcript가 구조화 intent 후보로 정리됨
- continue / pause / summary 계열 기본 voice intent가 action router에 연결됨
