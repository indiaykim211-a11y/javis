# Sprint 13: Visual Evidence Foundation

## 스프린트 목표

Phase 4 판단 엔진 위에 얹을 `시각 증거 입력 바닥`을 만듭니다.

## 포함 티켓

- `R5-001` capture target planner
- `R5-002` visual evidence packet builder
- `R5-003` browser and Codex observation prompts

## 상세 목표

### 1) 필요한 화면만 고르기

- Codex 창
- 브라우저 결과 화면
- 특정 오류 영역

중 무엇을 읽을지 우선순위를 정할 수 있어야 합니다.

### 2) 시각 입력을 구조화

- 현재 단계
- 기대 화면
- 캡처 대상
- 관찰 포인트

를 visual evidence packet으로 묶을 수 있어야 합니다.

## 완료 조건

- 관찰 프롬프트가 판정 중심으로 정리됨
