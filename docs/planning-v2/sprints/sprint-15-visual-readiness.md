# Sprint 15: Visual Readiness

## 스프린트 목표

시각 판단 흐름이 공격적으로 남발되지 않도록 캡처 안전 기준을 고정하고, Release 5를 반복 검증 가능한 상태로 마감합니다.

## 포함 티켓

- `R5-007` privacy and capture safety guard
- `R5-008` Release 5 smoke suite

## 상세 목표

### 1) capture safety

- 민감한 화면을 과하게 저장하지 않기
- 필요 없는 전체 화면 캡처를 줄이기
- evidence retention 기준 정리

### 2) release readiness

- continue / retry / ask_user가 visual evidence를 포함해 반복 검증 가능해야 합니다.

## 완료 조건

- 시각 판단 흐름을 5~10분 안에 다시 점검할 수 있음
