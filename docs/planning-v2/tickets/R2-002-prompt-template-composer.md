# R2-002: Prompt 템플릿 조합기

## 목적

Codex 시나리오 프리셋과 현재 프로젝트 컨텍스트를 합쳐서 `바로 쓸 수 있는 automation prompt 초안`을 만듭니다.

## 사용자 스토리

상단장님으로서, 마스터플랜과 정책을 다시 손으로 길게 옮기지 않고 prompt 초안을 빠르게 만들고 싶습니다.

## 범위

- 프로젝트 요약 / 목표 / 단계 / 정책을 prompt에 묶기
- 시나리오별 템플릿 조합
- prompt 새로고침
- 복사 / handoff 진입

## 완료 조건

- 전략 프리셋만 고르면 프로젝트 기준 prompt 초안이 생성됨
- 빈 프로젝트라도 최소 안내 문구가 포함됨
