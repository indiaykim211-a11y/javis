# R7-003 cloud trigger readiness tracker

## 목표

- cloud follow-up / trigger 성숙도를 무턱대고 가정하지 않고, readiness로 다룹니다.

## 범위

- supported / unknown / limited / ready 같은 상태 정의
- background follow-up 관점의 평가 기준
- 프로젝트별 운영 메모 연결

## 완료 조건

- cloud trigger readiness를 한눈에 분류할 수 있음
- readiness에 따라 integration mode가 달라질 수 있음
