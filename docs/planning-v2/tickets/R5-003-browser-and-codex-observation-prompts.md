# R5-003: browser and Codex observation prompts

## 목적

Codex 창과 브라우저 결과 화면을 `판정 중심`으로 읽게 하는 관찰 프롬프트를 만듭니다.

## 사용자 스토리

상단장님으로서, 화면 설명이 장황해지기보다 `정상인지 이상한지`를 빨리 알 수 있길 원합니다.

## 범위

- browser observation prompt
- Codex window observation prompt
- error / CTA / blank state / routing mismatch 관찰 지시

## 완료 조건

- 시각 관찰 프롬프트가 판정 목적에 맞게 짧고 일관되게 조합됨
