# R7-001 capability registry and integration mode selector

## 목표

- 현재 환경에서 가능한 deep integration 표면을 구조화해서 정리합니다.
- 그 결과를 바탕으로 추천 integration mode를 고릅니다.

## 범위

- capability registry 항목 정의
- recommended / selected mode 구분
- native 우선 원칙 반영

## 완료 조건

- capability를 카드나 표로 설명할 수 있음
- integration mode 추천 기준이 문서로 고정됨
