# R1-005: 프로젝트/세션 홈

## 목적

Control Center 안에서 현재 프로젝트와 최근 프로젝트를 자연스럽게 보고 이어서 쓸 수 있는 홈 화면을 만듭니다.

## 배경

현재 세션 저장 기능은 존재해도, 제품 관점에서 “지금 무엇을 이어서 쓰는지”를 보여주는 홈 경험이 약합니다.

## 사용자 스토리

상단장님으로서, 앱을 다시 켰을 때 현재 프로젝트와 최근 프로젝트를 쉽게 확인하고 이어서 작업하고 싶습니다.

## 범위

- 현재 프로젝트 요약 카드
- 목표 수준/현재 단계 표시
- 최근 프로젝트 목록
- 마지막 저장 시각
- 마지막 판단/캡처 요약

## 제외

- 장기 프로젝트 검색 시스템
- 원격 동기화

## 구현 메모

- 기존 세션 저장/복원 구조 재사용
- 최근 프로젝트는 정보량이 과하지 않게 요약 카드형으로 표시
- 팝업과 Project Home 간 이동 동선 고려

## 완료 조건

- Control Center 시작 지점으로 Project Home이 자연스럽게 동작
- 최근 프로젝트를 보고 바로 이어서 불러올 수 있음
- 현재 단계와 마지막 저장 상태를 한눈에 이해 가능
