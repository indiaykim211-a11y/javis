# R1-008: Release 1 스모크 스위트

## 목적

Release 1을 “형태만 바뀐 것 같은 상태”가 아니라 반복 확인 가능한 릴리즈로 마감합니다.

## 배경

팝업/Control Center/세션 복원처럼 UX 변화가 큰 릴리즈는 눈으로 보기만 해서는 누락을 놓치기 쉽습니다.

## 사용자 스토리

상단장님으로서, Release 1이 실제로 어시스턴트 중심 제품으로 바뀌었는지 빠르게 점검하고 싶습니다.

## 범위

- 첫 실행 체크리스트
- 팝업 표시 체크
- Control Center 진입 체크
- 프로젝트/세션 복원 체크
- 핵심 버튼 동선 체크

## 제외

- 장기 자동화 성능 검증
- 화면 인식 품질 검증
- 음성 기능 검증

## 구현 메모

- 사람이 빠르게 따라할 수 있는 시나리오 우선
- “제품 형태 전환”이 실제로 보이는지 확인하는 체크 포함
- 실패 시 어떤 화면을 보면 되는지 가이드 추가

## 완료 조건

- Release 1 핵심 흐름을 5~10분 안에 점검 가능
- 팝업/Control Center/세션 복원 흐름이 모두 체크리스트에 포함됨
- 이후 릴리즈 비교 기준으로 재사용 가능
