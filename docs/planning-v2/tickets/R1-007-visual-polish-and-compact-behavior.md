# R1-007: 컴팩트 비주얼/동작 다듬기

## 목적

작은 팝업이 실제 제품처럼 느껴지도록 시각 밀도와 동작 감각을 보정합니다.

## 배경

Release 1은 기능보다 첫인상과 사용 부담이 중요합니다. 컴팩트 팝업이 어색하면 전체 방향 전환의 의미가 약해집니다.

## 사용자 스토리

상단장님으로서, 팝업이 작아도 답답하지 않고 제품다운 완성도가 느껴지길 원합니다.

## 범위

- 팝업 크기/여백/타이포 조정
- 버튼 위계 정리
- 작은 해상도에서의 표시 보정
- 팝업/Control Center 전환 감각 조정

## 제외

- 대규모 브랜딩 리뉴얼
- 애니메이션 과다 도입

## 구현 메모

- 과한 툴 감성 제거
- 한국어 가독성 우선
- “작지만 고급스럽게” 원칙 유지

## 완료 조건

- 팝업이 작지만 부담스럽지 않음
- 버튼/카드 위계가 자연스러움
- 첫인상이 개발자용 제어판이 아님
