# R1-001: Assistant Popup 셸 창

## 목적

`javis`의 메인 창을 운영 콘솔이 아니라 작은 데스크톱 어시스턴트 팝업으로 전환합니다.

## 배경

현재 제품은 기능보다 메뉴가 먼저 보이고, 첫인상이 어시스턴트가 아니라 개발용 제어판에 가깝습니다.

## 사용자 스토리

상단장님으로서, `javis`를 실행했을 때 복잡한 콘솔 대신 작은 어시스턴트 팝업이 먼저 보이길 원합니다.

## 범위

- 팝업 전용 메인 창 도입
- 컴팩트 크기 기본값
- 항상 위 동작 옵션
- 헤더/본문/액션 바 기본 뼈대

## 제외

- 실제 판단 엔진 연결
- 화면 인식 결과 표시
- 음성 기능

## 구현 메모

- 현재 콘솔 창은 폐기하지 않고 Control Center로 이동
- 팝업과 Control Center를 별도 창 계층으로 분리
- 작은 창에서도 레이아웃이 무너지지 않도록 최소 크기 정의

## 완료 조건

- 앱 실행 시 팝업이 메인 창으로 보임
- 팝업에서 기본 카드/버튼 구조 확인 가능
- 기존 콘솔은 메인 첫화면이 아님
