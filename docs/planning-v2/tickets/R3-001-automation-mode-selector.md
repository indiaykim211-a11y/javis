# R3-001: automation mode selector

## 목적

현재 프로젝트에서 `no automation / thread automation / project automation` 중 무엇이 맞는지 먼저 추천합니다.

## 사용자 스토리

상단장님으로서, automation을 써야 하는지부터 바로 알고 싶습니다.

## 범위

- no automation 추천
- thread automation 추천
- project automation 추천
- 추천 이유 표시

## 완료 조건

- 현재 프로젝트 기준 추천 mode가 보임
- 왜 그 mode가 맞는지 짧게 이해 가능

## 의존성

- `R2-001`~`R2-006`

## 예상 난이도

- `M`
