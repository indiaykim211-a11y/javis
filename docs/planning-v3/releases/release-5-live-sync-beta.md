# Release 5. Live Sync Beta

## 한 줄 목표

`javis` web shell을 더 자주 켜놓고 믿고 쓸 수 있게, live sync와 운영 피드백 레이어를 완성합니다.

## 포함 기능

- workspace bundle endpoint
- activity feed API
- live sync heartbeat
- dirty-state guards
- activity timeline UI
- phase 5 verification

## 사용자 체감

- 화면과 세션 상태가 덜 어긋나 보입니다.
- 저장/전송/오류 흐름을 activity feed로 바로 읽을 수 있습니다.
- 편집 중인 draft가 자동 새로고침 때문에 갑자기 사라지지 않습니다.
