# Release 4. Control Deck Workspace Beta

## 한 줄 목표

`javis` web shell을 `보기 + 액션` 수준에서 `설정 + 편집 + 재진입`까지 가능한 실사용 베타로 끌어올립니다.

## 포함 기능

- control deck contract
- project / plan editor
- operations editor
- runbook / launch prompt panels
- prompt workbench
- recent project re-entry
- phase 4 verification

## 사용자 체감

- 이제 상단장님은 Tk 설정창보다 웹 셸을 메인 작업면으로 보기 시작할 수 있습니다.
- 프로젝트/단계/운영 메모를 웹 셸에서 바로 고치고 저장할 수 있습니다.
- 현재 프롬프트를 확인하고 다듬은 뒤 바로 `진행`으로 이어가는 흐름이 생깁니다.
