# Release 2. Assistant Surface Beta

## 한 줄 목표

`javis` 웹 셸을 진짜 운영 어시스턴트 첫 화면처럼 보이게 만든다.

## 포함 기능

- surface state contract
- action dock model
- timeline and prompt preview
- deck summary cards
- recent projects and signal panels
- refresh / offline states

## 시각 방향

- premium fintech dashboard density
- jarvis control room glow
- 더 또렷한 위계
- 읽기 쉬운 운영 카드

## 성공 기준

- 상단장님이 브라우저에서 봤을 때 “이제 첫 화면만으로도 운영 상태가 읽힌다”고 느낀다.
- Python 엔진의 상태 변화가 웹 셸 카드 구조에 자연스럽게 반영된다.
