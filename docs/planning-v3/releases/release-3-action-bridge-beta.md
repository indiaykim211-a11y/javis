# Release 3. Action Bridge Beta

## 한 줄 목표

`javis` web shell을 실제 운영 제어가 가능한 데스크톱 어시스턴트로 올립니다.

## 포함 기능

- action API contract
- Python bridge executor
- background auto loop
- wired action dock
- Control Deck drawer
- action feedback and verification

## 사용자 체감

- 이제 첫 화면 버튼이 장식이 아니라 실제 동작합니다.
- web shell만으로도 기본 운영 플로우를 반복할 수 있습니다.
- Tk 앱은 레퍼런스/엔진 역할로 한 걸음 뒤로 물러납니다.
