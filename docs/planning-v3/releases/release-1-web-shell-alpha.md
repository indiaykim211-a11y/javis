# Release 1. Web Shell Alpha

## 한 줄 목표

기존 Python 엔진 위에 `웹급 비주얼을 가진 첫 javis 셸`을 세운다.

## 포함 기능

- Electron 데스크톱 창
- React assistant surface
- Python snapshot API
- 프로젝트/단계/최근 프로젝트 요약 카드
- 핀테크 x 자비스 톤의 고급 시각 시스템

## 제품 느낌

- `운영 콘솔`보다는 `프리미엄 어시스턴트 패널`
- `개발툴`보다는 `운영 AI 인터페이스`
- 다크 네이비, 시안, 글로우, 오브, 데이터 카드 톤

## 성공 기준

- 상단장님이 첫 화면만 봐도 Tk 버전보다 확실히 더 제품 같다고 느낀다.
- 현재 Python 세션 데이터가 웹 셸에 반영된다.
- 이후 Phase 2에서 액션 브리지 붙일 기반이 깔린다.
