# H1-001 Electron + React Shell Scaffold

## 목표

`desktop-shell/` 워크스페이스를 만들고 Electron + React + Vite 실행 뼈대를 세운다.

## 완료 조건

- package.json 존재
- Electron main/preload 존재
- Vite React entry 존재
- dev/build 스크립트 정의

## 메모

- multi-terminal 실행을 기본으로 둔다.
- Phase 1에서는 orchestration 스크립트보다 안정적인 기본 구조를 우선한다.
